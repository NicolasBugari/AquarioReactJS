## **Para iniciar el proyecto realizar los siguientes pasos**

- Repositorio de Github.

  ```bash
  git clone https://github.com/NicolasBugari/AquiarioReactJS.git
  ```
- cd del proyecto.

  ```bash
  cd AquarioReactJS
  ```
- Instala dependencias.

  ```bash
  npm install
  ```
- Correr el proyecto.
  ```bash
  npm run dev
  ```
